# Prompt: Marktconsultatie-pagina

```
Bouw een Marktconsultatie-pagina op route `src/routes/(app)/projects/[id]/marktconsultatie/`.

De gebruiker kan hier een marktconsultatie voorbereiden en publiceren op TenderNed. De pagina heeft drie tabs:

1. **Consultatievragen** — Vragen opstellen voor de markt (toevoegen, bewerken, verwijderen)
2. **Marktpartijen** — Partijen uitnodigen voor de consultatie (voorlopig dummy data)
3. **Reacties** — Ontvangen reacties inzien

Daarnaast een rechterkolom met planningsdata (publicatie- en sluitingsdatum) en een live statusoverzicht.

De consultatie kan gepubliceerd worden zodra er minstens één vraag is ingevuld en één partij is uitgenodigd.

Bekijk de bestaande pagina's en componenten in het project voor stijl en conventies.
```
